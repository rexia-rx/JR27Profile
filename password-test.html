Password Toggle Test

Expected Behavior:

  • Password fields should have eye icons on the right side
  • Clicking the eye icon should toggle password visibility
  • Icon should change from open eye to crossed-out eye
  • Button should be properly positioned and clickable

Test Instructions:

  1. Type a password in the first field
  2. Click the eye icon to show/hide the password
  3. Verify the icon changes and password visibility toggles
  4. Test both password fields
  5. Check that the button is properly positioned
