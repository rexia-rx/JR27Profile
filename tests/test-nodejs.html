Node.js 环境安装指南

步骤 1: 安装 Node.js

请选择以下任一方法安装 Node.js：

方法 A: 官方安装包（推荐）

  1. 访问 nodejs.org
  2. 下载 LTS 版本（绿色按钮）
  3. 双击下载的 .pkg 文件
  4. 按照安装向导完成安装

方法 B: 使用 Homebrew

# 安装 Homebrew
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# 安装 Node.js
brew install node

步骤 2: 验证安装

安装完成后，打开终端并运行以下命令：

node --version
npm --version

如果显示版本号，说明安装成功！

步骤 3: 设置 React 项目

安装成功后，我们就可以设置 React 表单项目了。

# 进入项目目录
cd /Users/sac/Desktop/AUS\ IT/lesson1_code/part1

# 安装依赖
npm install

# 启动开发服务器
